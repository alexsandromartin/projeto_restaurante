# 🍽️ Restaurante Landing Page

Uma página web moderna e elegante desenvolvida para o setor gastronômico, focada em apresentar serviços e cardápio com um design responsivo.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as tecnologias fundamentais do desenvolvimento web:

* [cite_start]**HTML5**: Estruturação semântica do conteúdo[cite: 1, 2].
* [cite_start]**CSS3**: Estilização avançada com foco em organização modular[cite: 3, 4].
* [cite_start]**JavaScript**: Lógica de interação e comportamentos dinâmicos no front-end[cite: 3].

## 📦 Estrutura de Arquivos

O projeto está organizado de forma modular para facilitar a manutenção e escalabilidade:

* [cite_start]`index.html`: Arquivo principal contendo a estrutura da landing page[cite: 1, 2].
* [cite_start]`styles/styles.css`: Estilos globais e configurações principais do layout[cite: 4, 5].
* [cite_start]`styles/header.css`: Estilização específica e modular para o cabeçalho/menu[cite: 3].
* [cite_start]`javascript/script.js`: Arquivo responsável pela interatividade do site[cite: 3].
* [cite_start]`.vscode/settings.json`: Configurações de ambiente para o Live Server[cite: 1].

## 🛠️ Como Executar o Projeto

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/seu-usuario/restaurante.git](https://github.com/seu-usuario/restaurante.git)
    ```
2.  **Abra a pasta no VS Code.**
3.  **Inicie o Servidor Local:**
    * Este projeto está configurado para rodar via extensão **Live Server** na porta **5501**[cite: 1].
    * Clique com o botão direito no `index.html` e selecione "Open with Live Server".
